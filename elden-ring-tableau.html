Live Dashboard
A visual data guide to the Lands Between using Tableau.
This project explores data from Elden Ring, an open-world action RPG by FromSoftware known for its rich lore and deep mechanics and it uses data-driven visualizations to help players navigate the game’s complexity and make more informed choices based on their playstyle.
This was my final project for a Data Visualization course using Tableau — and as a big Elden Ring fan, I knew I wanted to dive into its rich, layered world. The game is packed with complex systems, from armor stats to boss locations, and I used this project to explore how Tableau’s advanced tools could make that complexity more navigable.
I focused on building interactive visualizations that help players (and curious designers) understand the game’s inner workings — like how items scale, where key challenges lie, and what makes each playstyle unique.
The dashboard is organized into thematic sections that guide users through Elden Ring’s complexity using interactive visualizations.
